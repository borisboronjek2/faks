PDF (Portable Document Format)
PDF je razvio Adobe 1993. godine kako bi omogućio dosljedno prikazivanje dokumenata neovisno o hardveru ili softveru. Kako bi se potakla njegova upotreba, Adobe je 2008. godine objavio PDF kao otvoreni standard kroz ISO organizaciju. Originalni PDF format bio je zaštićen patentima, no nakon što je postao otvoreni standard, licenca je postala slobodnija. ISO standardizacija omogućila je široku podršku bez direktnog kršenja patenata. PDF je normiran kroz ISO i široko se koristi u raznim industrijama. Podržava ga veliki broj softverskih alata, uključujući Adobe Acrobat, Microsoft Office, i razne web preglednici. Otvorena alternativa PDF-u uključuje format XPS koji je također standardiziran, ali PDF ostaje dominantan zbog svoje široke podrške i upotrebe.


SVG (Scalable Vector Graphics)
SVG je razvio W3C (World Wide Web Consortium) 1999. godine i prvi put objavio 2001. godine. Razvijen je kako bi se stvorio standardizirani format za vektorsku grafiku koji je lako integrirati u web stranice. SVG je otvoren format i nije podložan patentima. Sve specifikacije su dostupne javnosti, a razvoj je potpomognut zajednicom i organizacijama koje promiču otvorene standarde. Normiranje je izvršio W3C, što je dovelo do široke podrške među web preglednicima i alatima za grafički dizajn. SVG se koristi za grafiku na webu, animacije i interaktivne aplikacije, no podrška može varirati između različitih preglednika. Zatvorene alternative za SVG uključuju formate poput SWF, koji je bio popularan za animacije, ali više nije široko podržavan. Postoje i drugi vektorski grafički formati, ali SVG ostaje najrašireniji zbog svoje otvorenosti i interoperabilnosti.
