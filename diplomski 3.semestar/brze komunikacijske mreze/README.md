# BKM - izborni

Sam upis ovo predmeta je automatksi prolaz. Preporučio bi dolazak na uvodno predavanje. To uglavnom bude sat stand up komedije kod Bažanta. Na MI se ne mora izaći i čovjek samo upiše 20 bodova. Jedina obavzea oko predmeta je prijaviti ZI na koji se također ne izlazi i dobije se 20 bodova. Ništa drugo nema na predmetu ako planiraš samo proći.
